## Getting Started
### Pre-requisites
- Python 3.11 or later
- Poetry
  ```
    curl -sSL https://install.python-poetry.org | python3 -
  ```
### Installation
1. Create a virtual environment
    ```
    poetry shell
    ```
2. Install project dependencies
    ```
    poetry install
    ```
